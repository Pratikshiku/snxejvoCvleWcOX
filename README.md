## Personal Cloud Drive Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2f5c2466f7fb4e379606e29a8ea98389/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Blog Information
5. Forum
6. Announcement Information
7. Video Information
8. Book Information
9. Personal Center
10. Blog Information Collection
11. Background Management


**Administrator Functions:**
1. Login
2. Personal Center (modify password, personal information)
3. Administrator Management
4. Blog Information Management (blog information management, blog information collection management)
5. Basic Data Management (blog category management, announcement type management, video category management, book category management)
6. Forum Management
7. Announcement Information Management
8. Video Information Management
9. Book Information Management
10. User Management
11. Carousel Image Information
